v1.2.5 (beta25)

【新增】排班模式上線
- admin 後台幫您設好班次（早/中/晚/夜）後，App 上方顯示「✓ 上班中（離下班 X 分）」
- 不在班次時間顯示「⚠️ 不在班次時間，無法接單」+ 下次上班時間
- 不在班的時段不會收到訂單推送（後端自動過濾）
- 沒設排班的司機不受影響，行為跟以前一樣 24/7 待命

【新增】1+1 疊單（接單未下車前可接下一單）
- 行程快結束時（離目的地 < 2 公里）會收到「下一單已派發」小卡
- 卡片顯示新訂單的上車點 + 目的地
- 點「接受」→ 完成目前行程後自動接手新訂單，不用再操作
- 點「拒絕」→ 訂單退回派給其他司機

【新增】派單距離分批
派單從「綜合評分 Top-N」改成「距離遞增」：
batch 1 ≤2km → batch 2 ≤3km → ... batch 5 ≤15km
近的司機優先接到單，偏遠地區自動往外擴大半徑
