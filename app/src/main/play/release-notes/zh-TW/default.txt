GoGoCha v1.4.4 修「確認目的地」按鈕重複跳出 bug：

🐛 電話訂單按過「確認目的地」後到達階段又跳出來
   - 原因：confirmDestination 只更新本地 Order.destinationConfirmed
     一旦 WebSocket 重推訂單，server 回的 false 蓋掉本地 true
   - 修正：UiState 新增 confirmedDestinationOrderIds set 作 client-side override
     UI 雙重檢查 (Order.flag + Set)，按一次永遠生效（直到 app 關閉）
