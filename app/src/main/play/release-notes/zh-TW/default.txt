【新增】請客人重發上車位置（LINE 訂單專用）
接到 LINE 訂單到達現場找不到客人時：
- 訂單卡會出現「請客人重發上車位置」按鈕
- 按下 → 客人 LINE 收到通知 + 重新選位置的連結
- 客人在 LINE 內選好新位置 → 您的訂單卡上車地址即時更新
- 客人可同時選填手機號碼（LINE 訂單以前無法撥打）

【新增】LINE 訂單顯示付款方式 Tag
LINE 客人現在叫車時可選現金/愛心卡/敬老卡。
- 訂單卡會顯示對應的彩色 Tag
- 沒 Tag = 現金（不刷卡）
- 接到愛心卡/敬老卡訂單前可確認車上有刷卡機
